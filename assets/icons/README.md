# Icons (SVG reference)

These SVG files are **replaced by React icons** in the app. Do not import them directly.

Use the mapping in **`@/constants/assetIcons.ts`**:

| Asset file           | React icon (Ionicons)   |
|----------------------|-------------------------|
| Map pin / Map pin-1  | `location-outline`      |
| Ticket_alt / Two Tickets | `pricetag-outline` or `ticket-outline` |
| Calendar_check       | `calendar-outline`      |
| t-shirt (1) 1        | `shirt-outline`         |
| smiling-face / happy-face | `happy-outline`    |
| sad-face 1           | `sad-outline`           |
| confused 1           | `help-circle-outline`   |
| finish-flag 1/2      | `flag-outline`          |
| Navigation 2         | `navigate-outline`      |

Additional settings (createBusinessPost) use: `location-outline`, `navigate-outline`, `restaurant-outline`, `shirt-outline`, `musical-notes-outline`, `link-outline`, `cloud-download-outline`.
